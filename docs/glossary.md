# Glossary

**Archetype** — One of 10 brand personality categories used by BrandMind to classify brands

**CSR (Constraint Satisfaction Rate)** — Fraction of design constraints that pass verification

**WCAG** — Web Content Accessibility Guidelines; AA standard requires contrast ratio >= 4.5

**Coherence Score** — LLM judge rating (1-5) of how well fonts and colors match the archetype

**Revision Loop** — Iterative cycle where QC feedback triggers Agent 2 to regenerate

**Shared State** — LangGraph TypedDict passed between all three agents

**RAG** — Retrieval-Augmented Generation; grounding LLM outputs in real databases

**Heuristic Search** — Design rule retrieval with self-improving weights

**LangGraph** — Framework for building stateful multi-agent LLM applications

**Brand Kit** — Complete brand identity output: fonts + palette + tone descriptor
