# Brand Archetypes

BrandMind classifies brands into one of 10 archetypes:

| Archetype | Description | Typical Industries |
|---|---|---|
| Luxury | Exclusive, premium, timeless | Jewelry, Fashion, Hotels |
| Tech | Modern, precise, innovative | SaaS, Fintech, Cybersecurity |
| Playful | Fun, energetic, approachable | Kids, Gaming, Pets |
| Corporate | Professional, authoritative | Consulting, Legal, Finance |
| Minimal | Clean, refined, intellectual | Architecture, Design |
| Bold | High-energy, rebellious | Streetwear, Fitness, Esports |
| Organic | Natural, earthy, sustainable | Skincare, Food, Nonprofit |
| Artisan | Handcrafted, authentic, warm | Bakery, Coffee, Whiskey |
| Heritage | Classic, traditional, timeless | Whiskey, Museums |
| Youthful | Casual, vibrant, inclusive | EdTech, Fashion |
